html 기초

1.html이란?

html이란 hyper text mark-up language 의 약자이다.
즉, 자율적으로 웹 문서를 표시할 수 있도록 작성된 언어이다.
1990년에 팀 버너스 리가 과학자들과 연구 자료를 쉽게 공유하기 위해
개발한 것이 오늘 날 웹의 기초가 되었다.
html언어로 작성된 것은 브라우저로 번역하여 볼 수 있다.

2.xhtml,xml,html5,dhtml 이란?

xhtml : 좀 더 구조화된 html. 보다 엄격한 문법의 xml응용 언어.

xml : html보다 웹사이트 구축, 검색 기능이 향상된 언어.
sgml문서 형식을 따르고 있지만, sgml의 부분집합이다.

html5 : 가장 최신 버전의 html 표준.
애니메이션,그래픽,음악,영화 등을 모두 담을 수 있다.

dhtml : html, css, javascript를 조합하여 대화형 웹 사이트를 제작하는 기법.
경쟁기술로는 플래시, 자바

3.css란?

css는 cascading style sheets의 약자로, html,xhtml,xml같은 문서의 스타일을
꾸밀 때 사용하는 스타일 시트 언어이다.
글꼴이나 배경색, 너비와 높이 혹은 위치를 지정하거나
크기, 장치에 따라서 화면을 다르게 표시될 수 있도록 하는 것이 가능하다.
css는 1996년 w3c(표준 웹 문서 기관)가 도입했다.
이전에는 html 문서를 수정할 때 모든 문서를 하나씩 수정해야했지만
css의 등장 이후 문서의 내용과 표현을 분리하는 것이 가능하게 되었다.

스타일을 꾸미는 방법은 세 가지가 있다.

  1. 속성처럼 style 적용
  2. style tag 사용
  3. css파일을 별도로 만들어 html 문서에 연결

이 중 여러 문서를 수정하기엔 2,3번이 좋고, 그중에서도 3번이 가장 좋다.
